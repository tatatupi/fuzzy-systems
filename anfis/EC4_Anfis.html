Enunciado¶

Nesse exerccio vocs iro implementar o sistema nebuloso adaptativo visto em sala de aula para aproximar a funo sinc(x) no intervalo [0,2*pi].

Em anexo, segue a descrio do modelo, incluindo as derivadas parciais da funo de custo em relao aos parmetros do modelo.

Vocs devem entregar um arquivo fonte com a implementao do algoritmo. O arquivo deve treinar o modelo, plotar o resultado da aproximao e imprimir o RMSE.

Inicialmente, foram definidas funes para facilitar os clculos da sada do modelo:

  • Funo calcula_mu_w, que retorna:

    $\mu_{A\textit{lk}}(x_{il})=exp \bigg( -\frac{1}{2}\frac{(x_{il}-c_{lk})^2}{\sigma_{lk}^2} \bigg) \\w_k=\mu_{A\textit{1k}}(x_{i1})\mu_{A\textit{2k}}(x_{i2})...\mu_{A\textit{dk}}(x_{id})$

def calcula_mu_w(x,c,sigma): # x deve ser um ponto
    """x deve ser sempre uma amostra, i.e. um vetor de dimenso d"""
    mu = np.exp((-1/2)*(np.power(np.tile(x,(1,c.shape[0])).T-c, 2)/np.power(sigma,2)))
    w = pd.DataFrame(np.prod(mu,axis=1))
    return[mu,w]

  • Funo calcula_z, que retorna $y_k$ do artigo:

    $z_k=p_{1k}x_{i1}+p_{2k}x_{i2}+...+p_{dk}x_{id}+q_k$

In [223]:
def calcula_z(p,q,x): # x deve ser um ponto
    """x deve ser sempre uma amostra, i.e. um vetor de dimenso d"""
    z = pd.DataFrame((np.sum(p*pd.DataFrame(np.tile(x,(1,p.shape[0]))).T,axis=1)+q.T)).T
    return z

  • Funo calcula_y_hat, que retorna $\hat{y}_i$ do artigo:

    $\hat{y}_i=\frac{\sum_{k=1}^m w_ky_k}{\sum_{k=1}^m w_k}$

In [224]:
def calcula_y_hat(w,z):
    """x deve ser sempre uma amostra, i.e. um vetor de dimenso d"""
    y_hat = ((np.sum(z*w))/np.sum(w))[0]
    return y_hat

Em seguida, foram definidas funes para atualizao dos parmetros do modelo, conforme algoritmo do gradiente descendente:

$c_{lk} = c_{lk} - \alpha \frac{\delta J}{\delta c_{lk}} \\ \sigma_{lk} = \sigma_{lk} - \alpha \frac{\delta J}{\delta \sigma_{lk}} \\ p_{lk} = p_{lk} - \alpha \frac{\delta J}{\delta p_{lk}} \\ q_{k} = q_{qk} - \alpha \frac{\delta J}{\delta q_{k}}$

def atualiza_c(y_hat, w, mu, c, sigma, z, x, y, alfa):
    """x deve ser sempre uma amostra, i.e. um vetor de dimenso d"""
    aux = -2*(y-y_hat)[0]
    aux = aux*(z-y_hat)/np.sum(w)
#     aux = np.tile(aux,x.shape[1])*pd.DataFrame(np.tile(w,x.shape[1]))/mu
    aux = aux*mu*((pd.DataFrame(np.tile(x,(1,c.shape[0])).T-c)/np.power(sigma,2)))

    c_new = c - alfa*aux
    return c_new

def atualiza_sigma(y_hat, w, mu, c, sigma, z, x, y, alfa):
    """x deve ser sempre uma amostra, i.e. um vetor de dimenso d"""
    aux = -2*(y-y_hat)[0]
    aux = aux*(z-y_hat)/np.sum(w)
#     aux = np.tile(aux,x.shape[1])*pd.DataFrame(np.tile(w,x.shape[1]))/mu
    aux = aux*mu*(np.power(pd.DataFrame(np.tile(x,(1,c.shape[0])).T-c),2)/np.power(sigma,3))

    sigma_new = sigma - alfa*aux
    return sigma_new

def atualiza_p(y_hat, w, sigma, x, y, alfa, p):
    """x deve ser sempre uma amostra, i.e. um vetor de dimenso d"""
    aux = -2*(y-y_hat)[0]
    aux = aux*(w/np.sum(w))
    aux = np.tile(aux,x.shape[1])*np.tile(x,(p.shape[0],1))
    
    p_new = p - alfa*aux
    return p_new
In [228]:
def atualiza_q(y_hat, w, sigma, y, alfa, q):
    """x deve ser sempre uma amostra, i.e. um vetor de dimenso d"""
    aux = -2*(y-y_hat)[0]
    aux = aux*(w/np.sum(w))
    
    q_new = q - alfa*aux
    return q_new

Os hiperparmetros do modelo so declarados, sendo:

  • $m$, o nmero de regras do modelo
  • $alfa$, a taxa de aprendizagem
  • $num\_pontos$, a quantidade de pontos simulada, para a aproximao
  • $eps$, critrio de parada por convergncia
  • $epocas$, critrio de parada por tempo de simulao

Para determinar os valores otimizados dos hiperparâmetros, foi utilizado um algorítmo anexado ao final do trabalho.¶

Os parmetros iniciais do modelo so escolhidos de forma aleatria, de forma que:

  • $c_{lk}\sim U[min(x_l),max(x_l)]$
  • $\sigma_{lk}\sim U(0,1]$
  • $p_{lk}\sim U[-1,1]$
  • $q_{lk}\sim U[-1,1]$
In [231]:
# Inicializao dos parmetros
c = pd.DataFrame(np.random.uniform(np.min(x,axis=0),np.max(x,axis=0),(m,d))) 
p = pd.DataFrame(np.random.uniform(-1,1,(m,d)))
q = pd.DataFrame(np.random.uniform(-1,1,(m,1)))
sigma = pd.DataFrame(np.random.uniform(0,1,(m,d))) 
while 0 in sigma.values:
    sigma = sigma.replace(0,np.random.uniform(0,1))

Em seguida, o algoritmo executado, respeitando os critrios de parada. Os valores finais de $c, p, q$ e $\sigma$ configuram o modelo obtido e sero utilizados para estimar a funo sinc.

In [232]:
i = 0 # contador de iteraes
N = 0 # contador de pocas
dif = 1 # variao do vetor de parmetros
a=[] # armazena o valor de dif ao longo das iteraes
J=[] # armazena o erro estocstico ao longo das iteraes
start = time.time()

# variveis utilizadas para visualizar a variao dos parmetros ao longo das iteraes
c_m=pd.DataFrame(np.zeros((m,1)))
sigma_m=pd.DataFrame(np.zeros((m,1)))
p_m=pd.DataFrame(np.zeros((m,1)))
q_m=pd.DataFrame(np.zeros((m,1)))
mu_m=pd.DataFrame(np.zeros((m,1)))

while dif>eps and N<=epocas: # loop das pocas

    # Randomizando a ordem dos pontos
    index = np.random.permutation(len(x))
    xr = x.loc[index].reset_index(drop=True)
    yr = y.loc[index].reset_index(drop=True)

    while i<x.shape[0] and dif>eps: # loop dos pontos
        
        # Seleciona os pontos atuais
        xi = xr[i:i+1].reset_index(drop=True)
        yi = yr[i:i+1].reset_index(drop=True)

        # calcula mu, w, z e y_hat
        [mu,w] = calcula_mu_w(xi,c,sigma)
        z = calcula_z(p,q,xi)
        y_hat = calcula_y_hat(w,z)
    
        #atualiza os parmetros
        c_new = atualiza_c(y_hat, w, mu, c, sigma, z, xi, yi, alfa)
        sigma_new = atualiza_sigma(y_hat, w, mu, c, sigma, z, xi, yi, alfa)
        p_new = atualiza_p(y_hat, w, sigma, xi, yi, alfa, p)
        q_new = atualiza_q(y_hat, w, sigma, yi, alfa, q)
    
        #calcula a norma
        norma = [np.linalg.norm(c-c_new),np.linalg.norm(sigma-sigma_new),np.linalg.norm(p-p_new),np.linalg.norm(q-q_new)]
        
        #calcula critrio de parada
        dif = max(norma)
        a.append(dif) #armazena critrio de parada
    
        #atualiza os valores de parmetros
        if not math.isnan(dif):  #filtro para evitar erros de parmetros no numricos
            c = c_new
            sigma = sigma_new
            p = p_new
            q = q_new

        i+=1
        
        #armazena os parmetros para avaliar a variao ao longo do tempo
        c_m.loc[:,i]=c_new
        sigma_m.loc[:,i]=sigma_new
        p_m.loc[:,i]=p_new
        q_m.loc[:,i]=q_new
        mu_m.loc[:,i]=mu
        J.append(abs(y_hat-yi))
        
    N += 1
    i = 0

# armazena o tempo total de execuo do algortmo
end = time.time()

Escolha de hiperparâmetros¶

Para auxiliar na escolha dos hiperparmetros, o algortmo abaixo foi utilizado. Nele so realizados os seguintes passos:

  • os hiperparmetros so escolhidos aleatoriamente em uma lista de possveis valores definida anteriormente;
  • o algortmo de gradiente descendente aplicado para esses hiperparmetros;
  • caso um tempo limite seja atingido (30 segundos por poca ou 3 minutos por modelo), o grupo de hiperparmetros descartado,
  • em seguida o valor de RMSE calculado e caso seja inferior a um limite, os hiperparmetros e seus resultados so armazenados em uma tabela,
  • novos hiperparmetros so sorteados e o algortmo se repete at que um tempo limite seja atendido. Observando a tabela de resultados e os valores da lista de possveis hiperparmetros so atualizados, atingindo aqueles valores que no levam a bons resultados de RMSE. O algortmo ento executado novamente at que se chegue a hiperparmetros que obtenham um resultado satisfatrio em um tempo aceitvel.
